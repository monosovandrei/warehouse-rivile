# Склад Rivile GAMA

Внутренний веб-интерфейс для просмотра складских остатков из Rivile GAMA.

## Что делает

- Показывает только позиции, которые есть в наличии.
- Делит оборудование по типам на основе поля `Group` из Rivile.
- Группирует одинаковые детали с разными серийными номерами в одну строку.
- Суммирует остаток в колонке `Available`.
- Показывает закупочную цену в колонке `Purchase`.
- Показывает цену из листа `SUPERBASE` в колонке `Price`.
- Показывает отдельный раздел `Foreign stock` для зарубежных остатков поставщиков.
- Использует локальный суточный слепок, чтобы не дергать Rivile API при каждом открытии страницы.
- Использует отдельный суточный слепок цен из Google Sheet.
- Использует отдельный суточный слепок зарубежного прайса поставщиков, который можно забирать с внешнего supplier API.
- Читает страницы Rivile до конца списка; `RIVILE_PAGE_LIMIT` используется только как аварийный максимум.

## Локальный запуск

```bash
npm install
npm run dev
```

Открыть: `http://127.0.0.1:5173`

API работает на `http://127.0.0.1:5174` и читает настройки из `.env`.
