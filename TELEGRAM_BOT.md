# Локальный Telegram-бот: выручка и маржа

Бот запускается на локальном компьютере командой `npm run bot`. Сервер склада для этого не нужен.

## Что делает `/report`

- считает период с 1 числа текущего месяца по текущую дату включительно;
- берет продажи из Rivile `GET_I06_LIST`;
- включает только проведенные документы `I06_OP_TIP=51`, `I06_PERKELTA=2`;
- учитывает возвраты `I06_OP_TIP=52`, если они есть;
- считает количество как `1 проведенный заказ = 1 шт.`, а не как сумму товарных строк;
- включает доставку `9990006` в выручку и маржу менеджера/отдела;
- остальные услуги отделяет от оборудования;
- считает маржу как `I07_SUMA + I07_SAVIKAINA`, потому что себестоимость продаж в Rivile приходит отрицательным знаком.

## Методология отчета

Эту методологию не менять без отдельного решения:

- период считается включительно по датам отчета;
- продажи: `I06_OP_TIP=51`;
- возвраты: `I06_OP_TIP=52`, со знаком минус;
- документ должен быть проведен: `I06_PERKELTA=2`;
- строка доставки `I07_KODAS=9990006` входит в расчет менеджера и отдела;
- прочие услуги `I07_TIPAS=2` и остальные коды `999...` не входят в менеджерскую разбивку оборудования;
- маржа строки считается как `abs(I07_SUMA) - abs(I07_SAVIKAINA)` с учетом знака продажи или возврата;
- количество считается по уникальным документам: один заказ дает `1 шт.`;
- если менеджер не привязан к отделу через `REPORT_DEPARTMENT_MAP_JSON` или дефолтную карту, его заказ остается в общем итоге, но не попадает в отделы.

## Что сделать

1. В Telegram открыть `@BotFather`.
2. Выполнить `/newbot`.
3. Скопировать токен бота.
4. Добавить бота в нужную Telegram-группу.
5. В локальный `.env` добавить:

```env
TELEGRAM_BOT_TOKEN=123456789:replace_with_bot_token
REPORT_TIME_ZONE=Europe/Moscow
REPORT_CURRENCY=EUR
```

6. В папке проекта запустить:

```bash
npm run bot
```

7. В группе написать:

```txt
/report
```

## Ограничение доступа

Если бот должен отвечать только в одной группе:

1. Запустить `npm run bot`.
2. В группе написать `/chatid`.
3. Скопировать ID группы.
4. Добавить в `.env`:

```env
TELEGRAM_ALLOWED_CHAT_IDS=-1001234567890
```

После изменения `.env` перезапустить `npm run bot`.

## Проверка без Telegram

Чтобы проверить сам расчет по Rivile:

```bash
npm run bot:report
```

Эта команда просто печатает тот же отчет в консоль.
